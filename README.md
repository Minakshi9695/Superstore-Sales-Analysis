# Superstore-Sales-Analysis
This project showcases an end-to-end Data Analytics process, from data extraction to creating interactive dashboards.

## Project Overview
- **Objective**: Analyzed sales and profit data to identify business trends and optimization opportunities.
- **Tools Used**: 
    - **MySQL**: Used for data extraction and cleaning.
    - **Power BI**: Used for data visualization and dashboarding.

## Key Insights
- Analyzed regional sales performance and product category profitability.
- Identified that the "Furniture" category has a lower profit margin compared to others, indicating a need for operational optimization.
